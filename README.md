# OTel Collector running on Wasm+JS

View the slides:

https://kceu-2026-wasm-talk.pages.dev/

Setup steps:

1. Run `make serve` and say "yes" to any install prompts (e.g. npx wanting to
   install PNPM).
2. navigate to the URL in your browser and go through the slides. There's a demo
   of the Collector running toward the end.
